pub fn builder() -> TogglePropsBuilder<((), (), (), ())>

Create a builder for building ToggleProps. On the builder, call .checked(...)(optional), .disabled(...)(optional), .on_change(...)(optional), .class(...)(optional) to set the values of the fields. Finally, call .build() to create the instance of ToggleProps.
